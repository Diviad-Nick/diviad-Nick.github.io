极客大挑战 secret file

打开界面，是黑色色调上面有红色的字体，随便点一点没发现有什么登录框什么的

右键查看源代码，发现线索，进入第二个界面

在这个界面中，点击秘密的话，会提示你已经看完了，这时候需要抓包来看一下中间闪过的内容

中间闪过的内容提示，flag在secr3t.php文件中，访问这个文件，可以看到，提示flag在flag.php文件中，上方的代码显示，该页面存在include文件包含漏洞

直接file=flag.php发现并没有回显flag，说明直接包含是不行的，所以走另外一条路，php://filter包含

拿到base64加密过后的flag，解密后可以拿到flag
